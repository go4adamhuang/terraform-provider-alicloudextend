# alicloudextend_live_domain_url_signing

Manages the URL signing (URL authentication) settings for an ApsaraVideo Live domain. Set enabled = false to disable URL signing (auth_type = no_auth). Applies to both ingest and play domains.

## Example Usage

```terraform
resource "alicloudextend_live_domain_url_signing" "example" {
  domain_name = "live.example.com"
  enabled     = false
}
```

## Schema

### Required

- `domain_name` (String) The live domain name.
- `enabled` (Boolean) Whether URL signing is enabled. Set to false to disable URL signing (auth_type = no_auth).
