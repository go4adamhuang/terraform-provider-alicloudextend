# alicloudextend_live_domain_access_control

Manages the protocol blocking (access control) settings for an ApsaraVideo Live domain. Allows blocking RTMP and/or HLS playback protocols.

## Example Usage

```terraform
resource "alicloudextend_live_domain_access_control" "example" {
  domain_name = "play.example.com"
  rtmp_block  = "on"
  hls_block   = "on"
}
```

<!-- schema generated by tfplugindocs -->
## Schema

### Required

- `domain_name` (String) The live domain name.

### Optional

- `hls_block` (String) Whether to block HLS playback. Valid values: on, off. Defaults to off.
- `rtmp_block` (String) Whether to block RTMP playback. Valid values: on, off. Defaults to off.
